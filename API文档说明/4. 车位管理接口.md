### 4. 车位管理接口

#### 4.1 获取所有车位列表（包含占用状态）
**接口地址**: `GET /api/parking-space/list`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**: 无

**说明**: 
- `currentOccupiedStatus`: 0-空闲，1-占用中。同时检查 `space_occupy`（实际在场）和 `reservation`（有效预约占用）两张表。
- **可直接用此接口筛选空闲车位**：前端过滤 `status == 1 && currentOccupiedStatus == 0` 即可，无需单独调用其他接口。

**请求示例**:
```bash
curl -X GET "http://localhost:8080/api/parking-space/list" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": [
        {
            "id": 11,
            "parkAreaId": 1,
            "zoneId": 4,
            "spaceNumber": "B-001",
            "spaceType": 1,
            "status": 1,
            "latitude": 25.257826,
            "longitude": 110.122571,
            "bindUserId": null,
            "createTime": "2026-02-01T00:00:00",
            "updateTime": "2026-02-26T01:36:27",
            "currentOccupiedStatus": 1
        }
    ],
    "timestamp": 1777372563208
}
```

#### 4.3 创建车位
**接口地址**: `POST /api/parking-space`

**请求参数**:
```json
{
  "parkAreaId": 1,
  "zoneId": 3,
  "spaceNumber": "A-101",
  "spaceType": 2,
  "status": 1,
  "latitude": 39.9042,
  "longitude": 116.4074,
  "bindUserId": null
}
```

**参数说明**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| parkAreaId | Long | 是 | 园区ID |
| zoneId | Long | 是 | 分区ID |
| spaceNumber | String | 是 | 车位编号（如A-101），同一分区内唯一 |
| spaceType | Integer | 是 | 车位类型：1-固定，2-临时，3-访客，4-残障专用 |
| status | Integer | 是 | 车位状态：0-禁用，1-正常，4-故障 |
| latitude | BigDecimal | 否 | 纬度坐标（用于导航） |
| longitude | BigDecimal | 否 | 经度坐标（用于导航） |
| bindUserId | Long | 否 | 绑定用户ID（固定车位时关联park_user.id） |

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/parking-space" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -d '{
    "parkAreaId": 1,
    "zoneId": 1,
    "spaceNumber": "A-101",
    "spaceType": 2,
    "status": 1,
    "latitude": 39.9042,
    "longitude": 116.4074
  }'
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": {
        "id": 19,
        "parkAreaId": 1,
        "zoneId": 3,
        "spaceNumber": "A-101",
        "spaceType": 2,
        "status": 1,
        "latitude": 39.9042,
        "longitude": 116.4074,
        "bindUserId": null,
        "createTime": "2026-04-28T18:40:47.7272116",
        "updateTime": "2026-04-28T18:40:47.7272116",
        "currentOccupiedStatus": null
    },
    "timestamp": 1777372847763
}
```
