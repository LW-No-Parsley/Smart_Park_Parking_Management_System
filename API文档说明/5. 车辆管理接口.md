#### 5.3 创建车辆
**接口地址**: `POST /api/vehicle`

**请求参数**:
```json
{
  "userId": 1,
  "plateNumber": "京A12345",
  "isDefault": 1,
  "vehicleType": 1,
  "brand": "特斯拉",
  "color": "黑色",
  "status": 1
}
```

**参数说明**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userId | Long | 是 | 用户ID（关联park_user.id） |
| plateNumber | String | 是 | 车牌号 |
| isDefault | Integer | 否 | 是否默认车牌：0-否，1-是 |
| vehicleType | Integer | 否 | 车辆类型：1-小车，2-大车，3-新能源车 |
| brand | String | 否 | 车辆品牌 |
| color | String | 否 | 车辆颜色 |
| status | Integer | 否 | 车辆状态：0-禁用，1-正常 |

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/vehicle" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -d '{
    "userId": 1,
    "plateNumber": "京A12345",
    "isDefault": 1,
    "vehicleType": 1,
    "brand": "特斯拉",
    "color": "黑色",
    "status": 1
  }'
```

**响应示例**:
```json
{
  "code": 200,
  "status": true,
  "message": "操作成功",
  "data": {
    "id": 1,
    "userId": 1,
    "plateNumber": "京A12345",
    "isDefault": 1,
    "vehicleType": 1,
    "brand": "特斯拉",
    "color": "黑色",
    "status": 1,
    "createTime": "2024-01-01T10:00:00",
    "updateTime": "2024-01-01T10:00:00"
  },
  "timestamp": 1732982400000
}
```

#### 5.10 批量更新车辆状态
**接口地址**: `PUT /api/vehicle/batch-update-status`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ids | List | 是 | 车辆ID列表，用逗号分隔 |
| status | Integer | 是 | 车辆状态：0-禁用，1-正常 |

**请求示例**:
```bash
curl -X PUT "http://localhost:8080/api/vehicle/batch-update-status?ids=1,2,3&status=1" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**响应示例**:
```json
{
  "code": 200,
  "status": true,
  "message": "操作成功",
  "data": true,
  "timestamp": 1732982400000
}BUG："参数 'id' 类型错误，期望类型: Long"
```
