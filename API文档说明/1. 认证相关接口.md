### 1. 认证相关接口

#### 1.1 获取验证码

**接口地址**: `GET /api/auth/captcha`

**请求参数**: 无

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": {
        "captchaCode": "myCG",
        "captchaImage": "iVBORw0KGgoAAAANSUhEUgAAA....",
        "captchaId": "3d20dcd6d3234eb1b0041fce35035b2b"
    },
    "timestamp": 1777368137914
}"移除了captchaCode"
```

#### 1.2 用户登录（需要验证码）

**接口地址**: `POST /api/auth/login`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |
| captchaId | String | 是 | 验证码唯一标识 |
| captchaCode | String | 是 | 验证码 |
```

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "123456",
    "captchaId": "3d20dcd6d3234eb1b0041fce35035b2b",
    "captchaCode": "myCG"
}'
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": {
        "expiresIn": 3600000,
        "accessToken": "eyJhbGciOiJIUzI1NiJ9",
        "tokenType": "Bearer",
        "user": {
            "username": "admin",
            "phone": "19100000000",
            "email": "19100000@syan.top",
            "status": 1,
            "createTime": "2026-02-07T23:00:13"
        },
        "refreshToken": "eyJhbGciOiJIUzI1NiJ9"
    },
    "timestamp": 1777368217523
}
```

#### 1.3 用户注册（增强版，需要验证码）

**接口地址**: `POST /api/auth/register`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名 |
| password | String | 是 | 密码 |
| phone | String | 是 | 手机号 |
| email | String | 是 | 电子邮箱 |
| captchaId | String | 是 | 验证码唯一标识 |
| captchaCode | String | 是 | 验证码 |

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "newuser",
    "password": "Test@123",
    "phone": "13800138003",
    "email": "newuser@smartpark.com",
    "captchaId": "e8523b2d7c8647c3ba259518a6fbc025",
    "captchaCode": "8qAE"
}'
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "注册成功",
    "data": "",
    "timestamp": 1777368874937
}
```

#### 1.4 Token刷新

**接口地址**: `POST /api/auth/refresh`

**请求头**:
```
Authorization: Bearer {refreshToken} 
X-Access-Token: Bearer {accessToken} //旧的accessToken
```

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/auth/refresh" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." 
  -H "X-Access-Token: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**响应示例**:

#### 1.5 用户登出

**接口地址**: `POST /api/auth/logout`

**请求头**:
```
Authorization: Bearer <accessToken>
X-Refresh-Token: Bearer <refreshToken>
```

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/auth/logout" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "X-Refresh-Token: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "登出成功",
    "data": "",
    "timestamp": 1777369753911
}
```

#### 1.6 验证Token

**接口地址**: `POST /api/auth/validate`

**请求头**:
```
Authorization: Bearer {token}
```

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/auth/validate" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": {
        "valid": true,
        "message": "Token有效"
    },
    "timestamp": 1777369905628
}
```

#### 1.7 获取登录日志

**接口地址**: `GET /api/auth/login-logs`

**请求头**:
```
Authorization: Bearer {token}
```
**请求参数**:
- `userId` (可选): 用户ID，不传则查询所有用户
- `limit` (可选): 限制条数，默认10

**请求示例**:
```bash
curl -X GET "http://localhost:8080/api/auth/login-logs?userId=1&limit=10"
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": {
        "total": 10,
        "logs": [
            {

                "browser": "unknown",
                "os": "unknown",
                "status": 1,
                "failureReason": "",
                "loginTime": "2026-04-28T17:43:44"
            }
        ]
