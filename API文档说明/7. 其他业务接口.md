##### 7.1.3 创建进出记录
**接口地址**: `POST /api/access-log`

**请求参数**:
```json
{
  "parkAreaId": 1,
  "gateId": 1,
  "plateNumber": "京A12345",
  "vehicleId": 2,
  "accessType": 2,
  "imageUrl": "http://example.com/capture.jpg",
  "recognitionResult": 1,
  "accessTime": "2024-01-01T10:00:00"
}
```

**参数说明**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| parkAreaId | Long | 是 | 园区ID |
| gateId | Long | 是 | 道闸ID |
| plateNumber | String | 是 | 车牌号 |
| vehicleId | Long | 否 | 车辆ID |
| accessType | Integer | 是 | 进出类型：1-入场，2-出场 |
| imageUrl | String | 否 | 抓拍图片URL |
| recognitionResult | Integer | 否 | 识别结果：0-失败，1-成功 |
| accessTime | LocalDateTime | 是 | 进出时间（ISO 8601格式） |

**请求示例**:
```bash
curl -X POST "http://localhost:8080/api/access-log" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -d '{
    "parkAreaId": 1,
    "gateId": 1,
    "plateNumber": "京A12345",
    "vehicleId": 1,
    "accessType": 1,
    "imageUrl": "http://example.com/capture.jpg",
    "recognitionResult": 1,
    "accessTime": "2024-01-01T10:00:00"
  }'
```

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": {
        "id": 3,
        "parkAreaId": 1,
        "gateId": 1,
```
