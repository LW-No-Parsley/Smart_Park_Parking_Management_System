### 2. 园区管理接口

#### 2.1 获取所有园区列表
**接口地址**: `GET /api/park-area/list`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**: 无

**响应示例**:
```json
{
    "code": 200,
    "status": true,
    "message": "操作成功",
    "data": [
        {
            "id": 5,
            "name": "测试",
            "address": "桂林市临桂区",
            "totalSpaces": 1,
            "primaryAdminUserId": null,
            "latitude": 25.258288,
            "longitude": 110.124527,
            "businessHoursStart": "08:00:00",
            "businessHoursEnd": "22:00:00",
            "status": 1,

            "updateTime": "2026-04-27T23:48:03",

}
```

#### 2.3 创建园区
**接口地址**: `POST /api/park-area`

**请求头**:
```
Authorization: Bearer {token}
```

**请求参数**:
```json
{
  "name": "园区A",
  "address": "北京市朝阳区",
  "totalSpaces": 100,
  "latitude": 39.9042,
  "longitude": 116.4074,
  "status": 1
}
```
